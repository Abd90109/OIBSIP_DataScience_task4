# 📧 Spam Email Classifier

## Overview

This project focuses on building a machine learning model that can automatically classify emails as **Spam** or **Not Spam**. Spam emails are unwanted messages that often contain advertisements, phishing attempts, or malicious links. Detecting these emails automatically helps improve email security and user experience.

The model uses **Natural Language Processing (NLP)** techniques to convert email text into numerical features and then applies a machine learning algorithm to perform the classification.

---

## Objective

The main goal of this project is to develop a simple yet effective system that can analyze the content of an email and determine whether it is spam or a legitimate message.

---

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Natural Language Processing (NLP)

---

## Dataset

The dataset used for this project contains email messages labeled as either **spam** or **ham (not spam)**.

Typical columns include:

* **label** – Indicates whether the email is spam or not
* **message** – The actual text content of the email

---

## Project Workflow

1. Load and explore the dataset
2. Clean and preprocess the text data
3. Convert text into numerical features using TF-IDF
4. Split the dataset into training and testing sets
5. Train a machine learning model (Naive Bayes)
6. Evaluate the model
